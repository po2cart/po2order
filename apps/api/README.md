# @po2order/api

Hono API server for POtoOrder.

## Development

```bash
pnpm dev
```

API runs on http://localhost:3001

## Endpoints

See [docs/api-reference.md](../../docs/api-reference.md) for full API documentation.

## Tech Stack

- Hono (fast, lightweight)
- Zod (validation)
- Prisma (database)
